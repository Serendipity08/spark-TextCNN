项目概述

项目名称

情绪礼物

项目一句话价值

把你想说的话（文字或语音），在一个合适的情绪场景里，好好地送出去。

项目背景

在很多重要时刻，用户希望表达祝福、感谢、关心或情绪，但现有方式（文字、语音、模板祝福）往往存在以下问题：

文字表达情绪有限，容易显得冷淡

直接录语音容易紧张、卡壳，难以发送

模板祝福缺乏真实感和仪式感

本项目希望通过 AI + 场景 + 音乐 的方式，将用户的表达打包成一份可被认真接收的情绪礼物。

核心特征

支持文字祝福和语音输入两种创作方式

AI 对内容进行轻润色或生成，避免模板化

系统自动匹配情绪场景与背景音乐

生成私密的礼物接收页，通过链接或小程序分享

V1 范围（最小可行版本）

文字祝福 → 情绪礼物生成

语音输入 → 文案整理 → 情绪礼物生成（语音先模拟）

礼物生成、分享、接收、回执

不涉及登录、支付、社交关系链
